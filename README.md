##  Responsive Landing Page

###  Task Objective  
Create a **simple, responsive landing page** using only **HTML and CSS**.  
The page includes a **header, hero section, about, services, contact, and footer** — all designed to be clean, modern, and mobile-friendly.

###  Tools Used  
- **HTML5**  
- **CSS3 (Flexbox, Media Queries)**  
- **VS Code**  
- **Chrome Browser + Live Server**

###  Features  
- Fully **responsive layout** (mobile, tablet, desktop)  
- **Professional blue theme** for a modern look  
- Smooth **hover effects** and clean typography  
- **Sticky header** for easy navigation  

###  Sections Overview  
1. **Header** – Logo & navigation links  
2. **Hero Section** – Heading, paragraph & “Get Started” button  
3. **About** – Company introduction  
4. **Services** – Overview of what we offer  
5. **Contact** – Email for inquiries  
6. **Footer** – Copyright & social icons  

###  Outcome  
Understood and applied concepts of **HTML structure**, **CSS layout**, **Flexbox/Grid**, and **Media Queries** to create a modern, responsive design.
